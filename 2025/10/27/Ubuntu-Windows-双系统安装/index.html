Ubuntu/Windows 双系统安装

本文90%内容源于CSDN - Ubuntu/Windows双系统安装巨详细——全面解决各种问题，10%内容源于安装时自己的总结。

1. 制作 Ubuntu 启动盘

1.1 制作准备

  1. 你需要一个大于8GB的空U盘
    制作启动盘会格式化 U 盘，所有数据都会丢失！
  2. 首先在Ubuntu 官网下载系统镜像：
    推荐使用Ubuntu 22.04.5 LTShttps://releases.ubuntu.com/jammy/
  3. 然后下载制作系统盘的工具：
    推荐使用 Rufus 制作启动盘：https://rufus.ie/

1.2 制作步骤

  1. 打开 Rufus。
  2. 插入你的 U 盘。
  3. 确认设备选择了正确的 U 盘。
  4. 确认镜像选择了ubuntu-22.04.5-desktop-amd64.iso。
  5. 分区类型选择 GPT 。
  6. 目标类型选择 UEFI（非 CSM）。
  7. 点击开始。
  8. 选择以 ISO 镜像模式写入。
  9. 点击开始。
  10. 等待进度条完成即可。

2. 利用磁盘管理划分空间

  1. 按WIN + X选择磁盘管理。
  2. 选择一个空间较多的磁盘，右键点击选择压缩卷。
  3. 在压缩空间量中选择合适大小，用于安装 Ubuntu。

建议分配空间

使用场景 推荐空间大小
体验 Linux 30 GB
学习 ROS 80 GB 以上
深度学习 / 机器学习 100 GB 以上
软件开发 50 GB

可以看到有一块“未分配”的空间，下面还有一块启动盘的空间。

3. 安装前的准备工作

3.1 关闭 BitLocker

按下WIN + S搜索BitLocker，将其关闭。
否则可能导致电脑无法正常进入Windows系统。

3.2 关闭 Secure Boot

  1. 插入启动盘。
  2. 重启电脑。
  3. 出现Lenovo图标时，连按F2，进入BIOS界面。
  4. More Settings -> Security -> Secure Boot -> Disabled
  5. Exit -> Save and Exit

4. 安装与配置 Ubuntu

确保以上准备完成后：

  1. 重启电脑。
  2. 再次进入BIOS界面
  4. Try or Install Ubuntu。

4.1 开始安装

  1. 选择Install Ubuntu。
  2. 选择English/简体中文（按需选择）。
  3. 选择不连接网络（尚未更换源，下载会很慢）。
  4. 若空间有限可选最小安装。
  5. 注意！选择其他选项。

4.2 手动分区

选择你在 Windows 中预留的“未分配空间”。

启动分区：引导分区就是用来开机的，一般1G最多，有人给512MB，看你的需求

  • 大小：1024MB/512MB
  • 分区类型：主分区
  • 分区位置：空间起始位置
  • 用于：EFI系统分区

交换分区：swap就是类似电脑运存，一般是8G，好一点的就是32G，你电脑是多少你就给多少

  • 大小：8192MB/32768MB
  • 分区类型：逻辑分区
  • 分区位置：空间起始位置
  • 用于：交换空间

根目录：稍微分大一些，因为很多程序默认安装在根目录下

  • 大小：尽量分大一些
  • 分区类型：主分区
  • 分区位置：空间起始位置
  • 用于：Ext4日志文件系统
  • 挂载点：/

主目录

  • 大小：剩余所有空间
  • 分区类型：主
  • 分区位置：空间起始位置
  • 用于：Ext4日志文件系统
  • 挂载点：/home

其实你也可以选择将剩余空间全部挂载到根目录。因为Linux文件系统采用的是“使用分配”的策略。也就是说，你用了多少，就分配多少，没用的时候，剩下的空间就处于“待分配”的状态。

注意！
手动分区完成后，选择安装启动引导器的设备。
请选择刚分配好的EFI分区，千万别和windows的EFI分区搞错了，不然直接凉凉。
可以通过大小来区别两者，我们分配的大小会是1023MB。

4.3 继续安装

  1. 确认正确后选择现在安装。
  2. 选择Shanghai。
  3. 设置用户名和密码。
  4. 选择继续。
  5. 选择现在重启。

4.4 重启进入系统

  1. 拔掉启动盘。
  2. 按Enter。
  3. 选择Ubuntu。

欢迎来到 Linux 的世界！
