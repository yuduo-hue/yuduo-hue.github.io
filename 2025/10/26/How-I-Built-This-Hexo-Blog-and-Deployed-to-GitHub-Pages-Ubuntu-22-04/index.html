How I Built This Hexo Blog and Deployed to GitHub Pages (Ubuntu 22.04)

Created2025-10-26|Updated2025-10-26|Tutorials
|Post Views:

Goal: Create a Hexo blog on Ubuntu 22.04 and publish it to GitHub Pages using SSH, with simple commands to update the site whenever I write a new post.

1) Prerequisites

# Git (needed by Hexo deployer)
sudo apt update
sudo apt install -y git

# (Recommended) Install Node.js LTS via nvm
# If you already have Node and Hexo working, you can skip this block.
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts
nvm use --lts
node -v
npm -v

## 2) Install Hexo and create the site
`````bash
npm install -g hexo-cli
mkdir ~/blog && cd ~/blog
hexo init .
npm install
hexo s   # open http://localhost:4000 to preview

If the page opens locally, Hexo is installed correctly.

## 3) Prepare GitHub (User Site)

I’m using a user site so the URL is:
https://<username>.github.io

Create a repository named exactly <username>.github.io
(For me: yuduo-hue.github.io). In Settings → Pages, set the source to the same branch I’ll deploy to (I use main).

## 4) Set up SSH (no passwords)
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
# Press Enter for the default path, add a passphrase if you like

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

# Copy the public key and add it at GitHub → Settings → SSH and GPG keys → New SSH key
cat ~/.ssh/id_ed25519.pub

# Test:
ssh -T git@github.com
# Expect: "Hi <username>! You've successfully authenticated..."

## 5) Configure Hexo deploy

Install the deployer and set _config.yml (root file in the blog folder):

npm install hexo-deployer-git --save

# _config.yml (root)
url: https://yuduo-hue.github.io   # change to your username
root: /

deploy:
  type: git
  repo: git@github.com:yuduo-hue/yuduo-hue.github.io.git
  branch: main

##  6) First deployment
# Remove any cached deploy repo to avoid old HTTPS settings
rm -rf .deploy_git

hexo clean
hexo g
hexo d
