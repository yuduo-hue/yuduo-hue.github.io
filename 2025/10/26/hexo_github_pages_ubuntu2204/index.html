8) Common errors I met (and fixes)

  • ERROR Deployer not found: git → npm install hexo-deployer-git --save
  • Asks for GitHub username/password → switch to SSH in _config.yml
  • Still asks for password → rm -rf .deploy_git && hexo clean && hexo g && hexo d
  • Nothing changes after editing → hexo clean && hexo g before hexo d; check you edited the correct _config.yml (root vs theme)
