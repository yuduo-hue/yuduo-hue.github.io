二、创建 GitHub 个人主页仓库

  1. 打开 GitHub -> Create repository
  2. Repository name： <你的用户名>.github.io
    • 例如：yuduo-hue.github.io
  3. 勾选 Public
  4. 勾选Add a README file
  5. 点击Create repository

注意仓库名必须是用户名.github.io，不能是其他的

三、配置 SSH

1）生成 SSH 密钥

2）加载到 ssh-agent

3）复制 SSH 公钥

4）添加到 Github

  1. 打开 GitHub -> Settings -> SSH and GPG keys -> New SSH key
  2. Title: 任意
  3. Key type: Authentication Key
  4. Key: 刚才复制的 SSH 公钥

5）测试是否成功

出现 Hi <username>! You've successfully authenticated... 即成功。
